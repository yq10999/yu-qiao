興趣與技能

Python
Java

我從小就對於麻將感興趣，我認為它不僅僅是賭博的東西，他讓我的排列組合與邏輯概念更好，上大學後，也令我交到了許多的朋友，所以我覺得麻將只要不過量就不算是壞的習慣。

國小和國中的時候因為朋友的關係，所以我加入了籃球隊，一開始原本是想說陪陪朋友，但後來發現籃球是真好玩，每一次的比賽都讓我很喜歡，不管是贏還是輸，我很喜歡跟一群人合作完成一件事情的感覺。

Python對我來說真正有接觸到的時候是大一的時候，那時候對考試還是實作的時候都很緊張，不知道該怎麼寫，但上完課後，也發現Python是最基礎的，所以要好好學。

Java這堂課是接續著大一的Python延續下去的，因為是就讀資訊管理學系，所以我認為這兩個技能都很重要，之後會繼續努力學習，雖然大一的成績沒有到很好。

經歷

宿營心得

參加宿營是我大一就一直想做的事情，很謝謝有我的朋友陪伴我一起去，雖然練習的時候真的很累，很多事情要做，要練膽量跟音量，還要學會不可以有偶像包袱，但我很感謝當初有撐到最後的我，讓我有這麼一次特別的體驗，也很謝謝學長姐教會我這麼多的東西。

系烤心得

當初參加系烤是因為學長姐說要一起參加的，但真正參加後，我覺得他雖然沒有宿營那樣讓我很印象深刻，但也是不後悔，因為很多事情都在宿營的時候學過了，所以在做起來也相對的輕鬆。

系際盃

系際盃是班上同學慫恿我去的，當然也是因為我當時看完男生的系際盃後，內心澎湃，讓我一時衝動想去跟學姊一起打比賽，雖然隊裏的人我都不太認識，但他們人都很好，這次的比賽也讓我有重回國中的感覺。
